🚀 Getting Started

Welcome to OceanView Reservation System

The OceanView Reservation System is a comprehensive web application designed to streamline hotel reservation management. Whether you're an Administrator or Receptionist, this system provides powerful tools to manage your operations efficiently.

System Overview

Our system offers the following key capabilities:

  • Reservation Management: Create, view, search, and delete guest reservations
  • Room Availability: Real-time tracking of available rooms
  • Invoice Generation: Automatic invoice creation with print functionality
  • Reporting & Analytics: Revenue reports and occupancy statistics
  • Role-Based Access: Different permissions for Admins and Receptionists
  • Audit Logging: Comprehensive logging of all system activities

User Roles

👤 Administrator

Full system access including:

  • Manage all reservations (create, view, edit, delete)
  • Generate, view, and modify reports
  • Edit financial totals
  • Add and delete rooms
  • Create and manage Staff & Manager accounts

👤 Manager

Operational management including:

  • Create, view, and cancel reservations
  • Generate and view reports
  • Add and delete rooms
  • Print invoices
  • Cannot manage accounts or edit financials

👤 Staff

View-only access including:

  • View all reservations and bookings
  • View available rooms
  • Print invoices
  • Cannot add, edit, or delete reservations
  • Cannot generate reports or manage rooms

First Steps

  1. Log In: Use your username and password provided by your system administrator
  2. Dashboard: After login, you'll see the dashboard with key statistics
  3. Navigation: Use the top menu to access different sections
  4. Test Account: Admin: "admin" / "password123", Manager: "manager" / "password123", Staff: "staff" / "password123"

📖 User Guide

➤ How to Create a Reservation

Step-by-Step Instructions:

  1. Click on "New Reservation" in the navigation menu
  2. Fill in the Guest Information:
    • Guest Name (required, min 2 characters)
    • Phone Number (required, min 10 characters)
    • Complete Address (required)
  3. Select Reservation Details:
    • Check-In Date (cannot be in the past)
    • Check-Out Date (must be after check-in)
    • Room from dropdown (only available rooms shown)
  4. Review the Calculated Amounts:
    • Total nights are calculated automatically
    • Price per night is shown from selected room
    • Total amount is calculated (nights × price)
  5. Click "Create Reservation" to save
  6. Success message appears and redirects to View Reservations

💡 Tip: If no rooms are available, the system will display a warning message. You can use the "Clear Form" button to reset all fields at any time.

➤ How to View and Search Reservations

Viewing All Reservations:

  1. Navigate to "View Reservations" page
  2. All reservations are displayed in a table format
  3. Each row shows: Reservation #, Guest Name, Contact, Room, Dates, Amount
  4. Use pagination buttons at the bottom for large datasets (10 per page)

Searching Reservations:

  • By Reservation Number: Enter full or partial reservation number
  • By Guest Name: Search is case-insensitive, matches partial names
  • By Room Number: Enter exact room number
  • By Check-In Date: Find reservations from a specific date
  • By Check-Out Date: Find reservations until a specific date
  • Multiple Filters: All criteria work together (AND logic)

Action Buttons:

  • 👁️ View Details: Opens modal with complete reservation information
  • 🧾 View Invoice: Navigates to printable invoice page
  • 🗑️ Delete: Opens confirmation modal (Admin only)

💡 Tip: Click "Reset" to clear all filters and "Refresh" to reload data from the server.

➤ Generating Reports

Steps to Generate Reports:

  1. Go to the "Reports" page (Admin only)
  2. Select the Start Date and End Date for the report period
  3. Click "Generate Report" button
  4. The system displays:
    • Total Revenue for the period
    • Number of Reservations
    • Average Daily Rate (ADR)
    • Occupancy statistics
  5. Use "Print Report" button to print or save as PDF

💡 Tip: Reports are formatted for A4 printing with @media print CSS. Headers and navigation are hidden when printing.

➤ Managing Invoices

Accessing Invoices:

  1. From View Reservations page, click the 🧾 invoice icon
  2. Or navigate directly to Invoice page and enter reservation ID
  3. Invoice displays complete billing information including:
    • Guest details and contact information
    • Reservation dates and room details
    • Itemized charges (room rate × nights)
    • Total amount due

Printing Invoices:

  • Click "Print Invoice" button at top right
  • Browser print dialog opens automatically
  • Invoice is formatted for A4 paper
  • Print buttons and navigation are hidden in print version
  • Can save as PDF using "Save as PDF" option

💡 Tip: Invoices include company branding and contact information at the bottom.

➤ Using the Dashboard

Dashboard Overview:

The dashboard provides an at-a-glance view of your hotel operations:

  • Statistics Cards:
    • Total Reservations count
    • Active Reservations (checked-in guests)
    • Total Revenue (current period)
    • Current Occupancy Rate
  • Recent Check-Ins: Table showing today's check-ins
  • Quick Actions: Role-based shortcuts to common tasks
  • Occupancy Chart: Visual representation of room availability

💡 Tip: Dashboard automatically updates when you navigate back from other pages.

➤ Deleting Reservations (Admin Only)

Steps to Delete:

  1. Go to View Reservations page
  2. Find the reservation you want to delete
  3. Click the 🗑️ delete icon (visible to Admins only)
  4. A confirmation modal appears with warning message
  5. Click "Delete" to confirm or "Cancel" to abort
  6. Success message appears and table reloads automatically

⚠️ Warning: Deleting a reservation is permanent and cannot be undone. Make sure you have the correct reservation before confirming deletion.

💡 Tip: Only users with Administrator role can delete reservations. Receptionists will not see the delete button.

❓ Frequently Asked Questions

Q: How do I log in to the system?

A: Use your assigned username and password on the login page. Test accounts are: admin/admin123 for Administrator and receptionist/recep123 for Receptionist role.

Q: What if I forget my password?

A: Contact your system administrator to reset your password. Currently, self-service password reset is not available.

Q: Why can't I see the Reports or Delete buttons?

A: These features are restricted to Administrator role. If you're logged in as Receptionist, you won't have access to these functions. Check the role badge in the header to confirm your role.

Q: Can I edit a reservation after it's created?

A: Currently, the system doesn't support direct editing. To modify a reservation, delete the old one (Admin only) and create a new reservation with the updated information.

Q: How do I cancel a reservation?

A: Only Administrators can delete/cancel reservations. Go to "View Reservations", find the reservation, and click the 🗑️ delete icon. Confirm the deletion in the modal that appears.

Q: Where can I view the current occupancy rate?

A: The occupancy rate is displayed on the Dashboard and can be included in Reports. Only Administrators have access to the Reports page.

Q: What if no rooms are available for my selected dates?

A: The system will display "No rooms available" in the room dropdown. Try selecting different dates or check if existing reservations can be modified.

Q: How is the total amount calculated?

A: Total amount = Number of nights × Room price per night. The system calculates nights automatically based on check-in and check-out dates.

Q: Can I print invoices and reports?

A: Yes! Click the "Print Invoice" button on the invoice page or "Print Report" button on the reports page. These are optimized for A4 printing and can be saved as PDF.

Q: What browsers are supported?

A: The system works best on the latest versions of Chrome, Firefox, Safari, and Microsoft Edge. Internet Explorer is not supported.

Q: Why did I get logged out automatically?

A: For security, the system has a session timeout. If you're inactive for an extended period, you'll be automatically logged out and redirected to the login page.

Q: Can I use the system on mobile devices?

A: Yes, the interface is responsive and works on tablets and smartphones, though the experience is optimized for desktop use.

Q: What date format should I use?

A: The system uses your browser's default date format. Date pickers are provided for easy selection. Make sure check-out date is after check-in date.

Q: How do I search for a specific reservation?

A: Go to "View Reservations" and use any combination of filters: Reservation ID, Guest Name, Room Number, Check-In Date, or Check-Out Date. All searches are case-insensitive.

🔧 Troubleshooting

🚫 Problem: Cannot log in

Solutions:

  • Verify your username and password (case-sensitive)
  • Try test credentials: admin/admin123 or receptionist/recep123
  • Clear your browser cache and cookies
  • Make sure cookies and JavaScript are enabled
  • Contact your administrator if problem persists

🔄 Problem: Reservations not loading

Solutions:

  • Click the "Refresh" button on the page
  • Check your internet connection
  • Verify you're logged in (check session hasn't expired)
  • Clear any active filters that might be hiding results
  • Try logging out and logging back in

⚠️ Problem: "Permission Denied" error

Solutions:

  • Check your role badge in the header (Admin vs Receptionist)
  • Verify you have permission for that action (see User Roles section)
  • Reports and Delete functions are Admin-only
  • Try logging out and back in to refresh permissions
  • Contact administrator if you need elevated access

🖨️ Problem: Invoice or Report won't print

Solutions:

  • Make sure popups are allowed in your browser
  • Check your printer settings and connection
  • Try using "Save as PDF" option instead
  • Verify the page has fully loaded before printing
  • Use Print Preview to check formatting first

📝 Problem: Form validation errors

Solutions:

  • Read error messages carefully - they indicate what's wrong
  • Ensure all required fields (marked with *) are filled
  • Check date logic: check-out must be after check-in
  • Phone numbers must be at least 10 characters
  • Guest name must be at least 2 characters
  • Make sure a room is selected from the dropdown

🗑️ Problem: Delete button not visible

Solutions:

  • Verify you're logged in as Administrator (check role badge)
  • Delete function is not available to Receptionist role
  • Log out and log back in with admin credentials
  • Contact your administrator if you need deletion permissions

🔄 Problem: Dashboard not updating

Solutions:

  • Navigate away and back to refresh dashboard data
  • Press F5 or Ctrl+R to refresh the page
  • Clear browser cache if data seems stale
  • Check that you have active internet connection

⏰ Problem: Getting logged out frequently

Solutions:

  • This is normal security behavior after inactivity
  • Perform an action periodically to keep session active
  • Save your work frequently
  • Don't leave the page open without interaction
